Units

template<Magnitude auto M, typename U>
struct scaled_unit : public units::downcast_base<scaled_unit<M, U>>

A common point for a hierarchy of units.

A unit is an entity defined and adopted by convention, with which any other quantity of the same kind can be compared to express the ratio of the second quantity to the first one as a number.

All units of the same dimension can be convereted between each other. To allow this all of them are expressed as different ratios of the same one proprietary chosen reference unit (i.e. all length units are expressed in terms of meter, all mass units are expressed in terms of gram, …)

Template Parameters

  • M – a Magnitude representing the (relative) size of this unit

  • U – a unit to use as a reference for this dimension

template<typename Child, basic_symbol_text Symbol>
struct named_unit : public units::downcast_dispatch<Child, scaled_unit<mag<1>(), Child>>

A named unit.

Defines a named (in most cases coherent) unit that is then passed to a dimension definition. A named unit may be composed with a prefix to create a prefixed_unit.

Template Parameters

  • Child – inherited class type used by the downcasting facility (CRTP Idiom)

  • Symbol – a short text representation of the unit

template<typename Child, basic_symbol_text Symbol, Magnitude auto M, Unit U>
struct named_scaled_unit : public units::downcast_dispatch<Child, scaled_unit<M * U::mag, U::reference>>

A named scaled unit.

Defines a new named unit that is a scaled version of another unit. A named unit may be composed with a prefix to create a prefixed_unit.

Template Parameters

  • Child – inherited class type used by the downcasting facility (CRTP Idiom)

  • Symbol – a short text representation of the unit

  • M – the Magnitude by which to scale U

  • U – a reference unit to scale

template<typename Child, Prefix P, NamedUnit U>
struct prefixed_unit : public units::downcast_dispatch<Child, scaled_unit<P::mag * U::mag, U::reference>>

A prefixed unit.

Defines a new unit that is a scaled version of another unit by the provided prefix. It is only possible to create such a unit if the given prefix type matches the one defined in a reference unit.

Template Parameters

  • Child – inherited class type used by the downcasting facility (CRTP Idiom)

  • P – prefix to be appied to the reference unit

  • U – reference unit

template<typename Child>
struct derived_unit : public units::downcast_dispatch<Child, scaled_unit<mag<1>(), Child>>

A coherent unit of a derived quantity.

Defines a new coherent unit of a derived quantity. It should be passed as a coherent unit in the dimension’s definition for such a quantity.

Template Parameters

Child – inherited class type used by the downcasting facility (CRTP Idiom)

template<typename Child, DerivedDimension Dim, NamedUnit U, NamedUnit... URest>
struct derived_scaled_unit : public units::downcast_dispatch<Child, detail::derived_scaled_unit<Dim, U, URest...>>

A unit with a deduced ratio and symbol.

Defines a new unit with a deduced ratio and symbol based on the recipe from the provided derived dimension. The number and order of provided units should match the recipe of the derived dimension. All of the units provided should also be a named ones so it is possible to create a deduced symbol text.

Template Parameters

  • Child – inherited class type used by the downcasting facility (CRTP Idiom)

  • Dim – a derived dimension recipe to use for deduction

  • U – the unit of the first composite dimension from provided derived dimension’s recipe

  • URest – the units for the rest of dimensions from the recipe

template<Unit U, basic_symbol_text Symbol>
struct alias_unit : public U

An aliased named unit.

Defines a named alias for another unit. It is useful to assign alternative names and symbols to the already predefined units (i.e. “tonne” for “megagram”). A alias unit may be composed with a prefix to create a prefixed_alias_unit.

Template Parameters

  • U – Unit for which an alias is defined

  • Symbol – a short text representation of the unit

template<Unit U, Prefix P, NamedUnit AU>
struct prefixed_alias_unit : public U

A prefixed alias unit.

Defines a new unit that is an alias for a scaled version of another unit by the provided prefix. It is only possible to create such a unit if the given prefix type matches the one defined in a reference unit.

Template Parameters

  • U – Unit for which an alias is defined

  • P – prefix to be appied to the reference unit

  • AU – reference alias unit
