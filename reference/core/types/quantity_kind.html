Quantity Kind

template<Kind K, UnitOf<typename K::dimension> U, Representation Rep = double>
class quantity_kind

A quantity kind.

A quantity with more specific usage as determined by its kind. See https://jcgm.bipm.org/vim/en/1.2.html and NOTE 1 at https://jcgm.bipm.org/vim/en/1.1.html.

Template Parameters

  • K – the kind of quantity

  • U – the measurement unit of the quantity kind

  • Rep – the type to be used to represent values of the quantity kind
