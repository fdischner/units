Dimension one.

Dimension for which all the exponents of the factors corresponding to the base dimensions are zero. Also commonly named as “dimensionless”.

template<UnitOf<dim_one> U, Representation Rep = double>
using units::dimensionless = quantity<dim_one, U, Rep>
